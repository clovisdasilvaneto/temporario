Image Positioning

With any webpage comes images no matter how big or small they are! As you can see I have made a header image of which you are free to use! However, to position images within the main content area you do the following;

Image to the left

This image is floating to the left of the page. To do this put in between the image tag 'class="left"'.

Image to the right

This image is floating to the right of the page. To do this put in between the image tag 'class="right"'.

Image Centred

And finally this image is centred! this is done in two ways.

1) Put the image tag inside the <center></center> brakets.

2) .Put the image tag inside the <p align="center"></p> brakets.

Either way it should work!
