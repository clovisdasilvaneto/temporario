What is Autumn all about?

29th Sept 2005

This template was produced to look unique yet be easy to use. It is essentially a simple 2 column layout focusing on the following principles:

  • To have valid CSS and XHTML coding
  • Original appearance
  • Highly functional and accessible
  • Contribute something to the web design community who have taught me so much

Working with this template

27th Sept 2005

I have tried to make it easy to work with this template. I have annotated the CSS and XHTML

Editing can be done in a text editor such as Notepad or Crimson Editor and viewed in a browser (as the XHTML and CSS is fully commented) or you can use the excellent free NVU editor which can be downloaded here

Or of course you could use Dreamweaver, although sometimes the preview displays quite strangely
