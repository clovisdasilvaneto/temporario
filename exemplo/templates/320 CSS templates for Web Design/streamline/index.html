About streamline

streamline is intended to be just that - a streamlined template that utilizes a sleek look that focuses on the use of shades of gray (one of my favorite colors, by the way). This page is well suited to use alpha transparency, but since it is not supported by "certain" browsers (see below), I have removed it from this template. To see this template at its best, visit my personal site, which uses alpha transparency.

Specifics Regarding the Template

Eternal Spite Upon Internet Explorer!

streamline used to feature .png images (pronounced "ping") that allows it to use transparency features not available to any other standard image format (.gif and .jpg). Internet Explorer, however, has never implemented the format correctly. Therefore, to remain compliant with IE 5.5/6's shoddy standards, the images were removed. This template is also renderable in Firefox 1.0.7 / 1.5 and Opera 8.51

Usage Agreement

You may use this template for whatever purpose. Even the original streamline logo is available for your use.

Accessibility

This design is largely accessible. Markup provides easy use for non-visual browsers (braille readers, screen readers, text-only browsers) and low-bandwidth connections (handhelds, such as PDAs). Styling with CSS allows the site to appear streamlined for visual browsers. It should be noted that just because of this does not necessarily make it accessible - it has not been tested according to the w3c's Web Content Accessibility Guidlines, nor the section 508 guidlines.
