About This Template

This layout is valid XHTML 1.1 and Valid CSS! A font named "Eurostile" is used in this layout. If the users computer doesn't have it, it will revert to a generic sans-serif font.

Browsers

This layout has been tested extensivly in all major browsers (and by extensive, I mean a 10 second look to see if it was the same). Opera 8.5, Firefox 1.5 and IE6 all render the page the same, they render the fonts differently but that's how the browsers were built. IE makes sans-serif smaller in width i've noticed.
