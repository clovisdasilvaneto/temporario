Gamer 01-dk

This template is "Gamer 01-dk", the first in a series of video game-themed templates for Open Source Web Design. I was inspired to do this by mikeystan's Nintendo template (which also was supposed to be a part of a series, by the way). I fancy myself a gamer, as I have a NES, Super NES, and a Game Boy Advance SP, to name a few. Of my games I treasure my "classic" games for the NES and Super NES.

But About the Page?...

The page validates as XHTML Strict 1.0 and validates as CSS. It has been tested in Firefox 1.0.7, Opera 8.51, and Internet Explorer 6.0. Guess which browser displays this the worst? (I'll give you a hint: I've put them in order from best to worst.) As noted in the CSS file, you should keep a doctype on the page for the best possible rendering. (Reasons for this are put forth in the CSS code.)

It was coded by no other than me, Generalissmo. This, I've got to say, trumps all my other designs (as of this writing) when it comes to having entertainment value. If you plan to use this page, send me the link, I really want to see it in action.
