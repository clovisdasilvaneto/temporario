Introduction to Contented2

Contented2 is the second open source template from Contented Designs. The breadcrumb navigation that shows users where they are can be omitted if your website's information structure has too few levels.

Contented2 is valid CSS and valid XHTML 1.0 strict. It's been tested for consistent appearance on:

  • Internet Explorer 6.0.2900.2180
  • Firefox 1.5.0.1
  • Opera 8.53

Current version: Contented2 Version 1.0 (April 14, 2006)

Using this Design

Contented2 is an open source template. You're free to modify it and use it for any purpose without cost or obligation. We prefer that you leave the link to our website in the footer but it's not required.

If you have comments or questions, please contact us at Contented Designs. Thanks!
