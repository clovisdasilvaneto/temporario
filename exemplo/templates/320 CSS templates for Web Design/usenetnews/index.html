My First Design

This is my first design submitted to OSWD. It's a black and white css template created for Usenet News and designed to:

  • Load fast
  • Look presentable on my Verizon PocketPC

Why such a simple design? Because the website it was originally designed for needed to work on a broad number of displays. The webmasters expect traffic from web browsers, phone browsers and PDA browsers alike.

So if you find yourself needing a fast loading, image free, cross platform design then consider "Usenet News" :). You can use this template for whatever, all I ask is if you want please keep a link to my website and email me, I really enjoy seeing my templates in action.
