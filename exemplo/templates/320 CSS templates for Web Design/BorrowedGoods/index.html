This is an update of my first (and currently my only) submission to OSWD.org. It resizes pretty nicely, and requires no images or tables. The "Site Name | Directory | Page" in the top left corner of the page now corresponds with h# tags (h1 | h2 | h3), and the unnecessary divs from the original version have been removed. I've also included a print style sheet, which hides everything but your content.

I'd love for you to keep "Designed by quixotic" on the page, but you most certainly don't have to. As long as you freely share any changes you make to this template, you're free to use it without any accreditation.
