about the design

This design was created by acousticsam for OSWD. This layout is open source.

My first attempt at an image-based layout, Penchant For Photos is ideal for a photo gallery or similar website. I have included a sample gallery; view it via the navigation menu. However, all the other links are not functional.

about the designer

I have been building web sites since 2001, starting out with basic HTML 4.01 and table-based layouts. I have progressed into the realm of CSS and XHTML1.1 in recent years, favouring DIVs rather than tables. I have yet to learn JavaScript, but that is the next big step in my web design career.

I code for the fun of coding, building web pages on a whim and testing out various ideas. I try to focus beyond aesthetics and write clean, easy-to-read, standards-compliant code with lots of helpful comments.

design specifications

Penchant For Photos was tested in FF1.0.7 and IE6. It validates as standards-compliant for XHTML 1.1 and CSS.

All images are free to use without restriction. The background image was created with this background image maker powered by Ruby on Rails. The background image for the content box, as well as the images in the sample gallery, are taken from the Stock.Xchng. The navigation icons are also free to use.
