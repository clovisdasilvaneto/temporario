This is Syncostyle
Posted 31st December 2005 at 5:14PM - 0 Comments - 2 Reads

It's synchronised. It's stylish. It's Syncostyle.

Well, at least that's the idea. I tried to keep it plain. Plain is good. Pure white (almost). The layout is pretty clean, maybe for a simple blog site. The only image is the header, which is just there for added style ... ;-)

Now the code side... it works in Opera, Firefox, and IE. It's valid XHTML 1.0 Transitional, and almost valid CSS. There is one error because of "overflow-x", but that's needed for IE, so not much can be done about it. It is totally CSS and div based, with tables banished to the nether regions.

If you use it on your site, please email me, and tell me the site it's being used on ... I like to know who's using my designs. And don't remove the "Designed By" part in the footer. Feel free to use the design as you wish, just don't remove the footer line, or take credit for my work... that's all. I don't want to sound too mean... ^_^
