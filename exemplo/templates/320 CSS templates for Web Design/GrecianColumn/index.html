Grecian Column Theme

I have been working on this for a while. It is a simple 800 by 600 layout that with a little effort can be modified for higher or lower resolution.

You should be able to widen the column to allow for 160 pixels as well as stretch the top column a little to allow for a banner ad too!

You can have a vertical menu along the columns or you can hang signs or anything else you want.

I sincerely hope you enjoy this template. If you use it in production please let me know by emailing me at admin@idavista.com. I would love to see it in use.

I hope to have more templates soon. Enjoy the template.

You can hang signs from the columns!
