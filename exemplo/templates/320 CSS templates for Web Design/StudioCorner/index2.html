Wellcome

This page is third design after Calypso Gallery and BlueNight in OSWD. It is again built by XHML 1.0 Strict and CSS 2.0 rules. Page text code is UTF-8.
 
The design has main two columns. Fixed left column (you can do more/less wide), resizable right column according to your browser's size. The interesting feature about this site is that it doesnt use any tables. Text boxes were made different size in css as a class. Organized and adequately formatted "DIV" tags are very useful!
