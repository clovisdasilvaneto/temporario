Welcome to Lime-Light

Green, lean, mean.... welcome to my first Night Club / Rave template. This fixed template is 100% valid CSS/XHTML, and completely un-compliant with everything else. If you're colorblind you won't like this site. Hopefully everybody else will.

Images Galore

This template stretches the 25KB OWD limit to the max. Images form a central part of the visual appeal of this template, but I worked very hard to keep them seperate from the layout.

The following images are used:

  • zigzag - background image. Can be turned on or off, or easily modified
  • gradients -- vertical gradients going from light to dark and vica versa. I use these in the background of all boxed div's to give them shape
  • banner - the banner image can make a big difference in the site
  • arrow - there are red, yellow, and green images used in the lists and for the menu buttons. I've included flashing and static gif images of each
  • lemon/lime/orange slices - these are included to demonstrate how images can spruce up a page.

Alternate Layouts

I struggled with what layout I should put into this template -- should it be classic 2-column, 3-column, or what?

Finally, I came to the conclusion that a real website often needs many layouts in the same template... so I put them all in here.

Most links in the template are dummy links, but the nav links ("Home", "About", etc) are links to the alternate layouts. They are:
