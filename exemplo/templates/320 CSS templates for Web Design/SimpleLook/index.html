The world's newest web template based on an e-mail client

About this Template

This template is a visual re-construction of MS Outlook 2003 and the Firefox Theme "Outlook 2003 Blue".

This template is 100% valid CSS/XHTML-Strict and a few gradient images -- the only known bugs are the fact that internet explorer ignores the CSS hover selector on the images for the lower left buttons, and of course that IE doesn't do position:fixed either. So, in IE the left side scrolls also, but in Firefox it stays put and only the right side scrolls.

This image was "borrowed" from Andreas Viklund's andreas01 template.

Why SIMPLELOOK?

Visual effect!

Now I know that half the known world (including yours truly) isn't a big fan of M$ or Outlook for that matter, but I do like the widgets, and the whole look and feel is cool. So, I wanted to make a web template just like it.

Table Support Included

It's been said probably 1,000,000 times by a billion people -- tables aren't evil. Tables are wonderful -- for tabular data.

This layout uses CSS and HTML for everything, but I wanted to throw in some table support because I use tables all the time and they're quite useful -- for tabular data, such as rows coming out of a database.
