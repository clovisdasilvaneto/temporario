GENERAL CONSIDERATIONS

Default fonts are defined for each of the major regions of the webpage, that is, header, left navigation bar, content, right sidebar and footer. To make the change of default font-sizes (and partially colors) more convenient like a one-button approach, the font definitions have been broken out into separate generic CSS-classes. The font-sizes for each of the regions are then defined in percentages of the default font-sizes specified in the generic CSS-classes. Default font-type is Trebuchet MS for best looks.

How it works

  • Generic font CSS-classes:
    - Header top navigation: .nav-global-font
    - Header main navigation: .nav-main-font
    - Left side navigation: .nav-sub-font
    - Content area: .content-font
    - Sidebar: .sidebar-font
    - Footer: .footer-font
  • Method: Applied at the main HTML-tag for each region.
