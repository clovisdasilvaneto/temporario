GENERAL CONSIDERATIONS

The GoFlexiblePro color scheme is divided into background colors and text colors. Each of these groups can be applied individually or simultaneously to HTML-tags by using generic CSS-classes ("generic" = classes that have only one property defined). When the generic CSS-classes are applied, the default settings are overruled. In this way we have a powerful global template with the combination of flexibility to adapt local sections of the webpage without any need to modify the CSS-code. This is convenient for beginners.

GoFlexiblePro contains five color groups: grey, yellow, blue, green and red. For each of these groups there are ten shading levels offered. If you prefer other color schemes, just go to the CSS-code and define the generic classes according to your tastes or needs.

BACKGROUND COLORS

How it works

  • Basic color set:
    - White
    - Black
    - Grey (Increments of 5-10%)
    - Yellow (based on "Pantone Yellow C", increments of 10%)
    - Blue (based on "Pantone 2726C", increments of 10%)
    - Green (based on "Pantone 369C", increments of 10%)
    - Red (based on "Pantone 179C", increments of 10%)
  • Generic CSS-classes:
    - White: .bg-white
    - Black: .bg-black
    - Grey: .bg-greyNN (NN = number between 01 and 10)
    - Yellow: .bg-yellowNN (NN = number between 01 and 10)
    - Blue: .bg-blueNN (NN = number between 01 and 10)
    - Green: .bg-greenNN (NN = number between 01 and 10)
    - Red: .bg-redNN (NN = number between 01 and 10)
  • Method: Apply generic CSS-classes to HTML-tags
