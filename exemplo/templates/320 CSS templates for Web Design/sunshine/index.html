Images

Any website without images somewhere tend to be a bit boring so as always I have added the three image rules;

  1. To align an image to the left add 'class="left"' In the image tag
  2. To align an image to the right add 'class="right"' In the image tag
  3. To align an image centre put the image tag inside either the <center></center> tags or <p align="center"></p>
