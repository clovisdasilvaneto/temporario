About this template

This is my first submission to OpenWebDesign.org and OpenSourceWebDesign.org. I felt the need to contribute to these communities so I used my website's layout and changed the colors and the logo. However, sytles for block quotes, forms, etc. ate not implemented (I don't use them on my site). Sorry for the inconvenience. You can awlays send me an e-mail if you need some help.

The header image was taken from DesignsByDarren.com. I reduced it a little in dimensions and file size.

You can use DimP_01 for any purpose but please send me an e-mail. I just want to see my work in action. Feel free to adjust this template to your taste but please don't resubmit it with minor chages.
