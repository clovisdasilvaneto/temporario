I designed Daffodil to be used with a screen width of 1,024 pixels. For the OSWD.org submission it is set to an 800 pixel width. The width can be easily changed in the CSS file. Instructions are in the comments.

This design does limit the amount of text you can use per page but you can use this design as a model to make your own page with your own photo. A photo that fades to white or other color would work best so your visitor can scroll down the page to view the whole page without the photo abruptly stopping.
