Uncommon Web Design

Web design these days is usually a disaster! With completely invalid html/xhtml and CSS that makes your head explode, it's tough to find a really good design out there. I have tried here to mix a little tech with some old school to find a sweet harmony that is truly Uncommon web design.

The design is written enitrely from scratch (though inspiration was take from other sites, not code) in XHTML 1.0 Transitional and CSS 2. Both the style sheet and the actual page have been validated using the W3 validation services and passed 100%

Note: An alternative "Bread Crumb" style navigation bar can be found in the css source code and all on ehas to do is uncomment it. It may better suit your needs.
